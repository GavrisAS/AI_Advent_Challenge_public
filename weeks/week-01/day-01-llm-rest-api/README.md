## Исходное условие

🔥 День 1. Первый запрос к LLM через API

СНАЧАЛА ПОСМОТРИТЕ ВИДЕО

Напишите минимальный код, который:

- отправляет запрос в LLM через API;
- получает ответ;
- выводит его в консоль или простой интерфейс (CLI / Web).

Результат: код, который отправляет запрос в LLM через API и получает ответ.

Формат: видео + код.

Видео к заданию: https://disk.yandex.ru/i/iDqQAsdBVg8y7g

## Цель задания

Отработать минимальный REST-вызов LLM API: подготовку HTTP-запроса, передачу ключа, разбор ответа и вывод результата в CLI.

## Реализация

В `snapshot/LLM_REST_API.py` реализован интерактивный CLI-чат с DeepSeek-compatible API. Скрипт загружает переменные окружения из локального `.env`, хранит историю сообщений текущей сессии и поддерживает команды `/help`, `/reset`, `/exit`.


## Выводы

Минимальный вызов LLM через REST API требует отдельной обработки конфигурации, ошибок HTTP и структуры `messages`. Даже простой CLI быстро начинает нуждаться в истории сообщений и базовых командах управления сессией.
