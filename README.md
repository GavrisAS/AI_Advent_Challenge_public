## Как запускать задания

Каждый день содержит собственный README с инструкциями запуска.
Для дней с кодом внутри папки дня есть самодостаточный `snapshot/`.

## Видео-отчёты

Видео-отчёты выполнения заданий доступны по Google Drive-ссылкам в README соответствующих дней.

## Примечание

Учебные конспекты, приватные заметки, memory-bank и внутренние материалы курса в публичный репозиторий не публикуются.
